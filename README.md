# Freelance Invoice Generator

A Python command-line application that generates freelance invoices based on region-specific tax rules.

## Features
- Supports EU (VAT 20%) and USA (Tax 5%)
- Service fee applied for high-value invoices
- Input validation for names and numbers
- Exception handling for safe user input
- Clean, readable invoice output

## Technologies Used
- Python 3
- Core concepts: functions, conditionals, loops, exception handling
